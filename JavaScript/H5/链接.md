# a

```html
<!-- 
a 标签为双标签，需要有首尾标签。

  href="url|锚点元素id|js表达式|相对路径文件" 规定链接指向的页面的 URL。#或javaScript:;不跳转
  download="filename" 规定被下载的超链接目标。
  hreflang="language_code"  规定被链接文档的语言。
  media="media_query"  规定被链接文档是为何种媒介/设备优化的。
  referrerpolicy="" 规定当用户单击超链接时要发送哪些引荐来源信息。
    no-referrer 
    no-referrer-when-downgrade 
    origin 
    origin-when-cross-origin 
    same-origin 
    strict-origin-when-cross-origin 
    unsafe-url
  ping="list_of_URLs"  规定以空格分隔的 URL 列表，当点击链接时，浏览器将发送带有正文 ping 的 post 请求（在后台）。通常用于跟踪。
  target 规定在何处打开链接文档。
    _blank 新窗口
    _parent 父框架
    _self 默认,此窗口
    _top 整个窗口
    framename 指定窗口
  type="MIME type" 规定被链接文档的的 MIME 类型。
  rel="简介"  定义当前文档，主要用于搜索引擎搜集网页内链接的集合以及链接与链接之间的关系，没有实际功能用途。
  title="鼠标悬浮显示的提示信息"
-->
<a href="http://ip:port/path">内容 </a>
```

## 锚点定位

```html
锚点定位：让页面跳转到的位置，就是锚点。 锚记 和 指向锚记的链接 1. 使用 a 标签作为锚记和链接， name定位，href与name对应，herf属性加上#,herf="#name"
<a href="#mao">点击此处到目标位置</a>
<br /><br />
<a name="mao">目标位置</a>

2. 使用 a 标签作为链接，使用其他元素（建议使用块元素）作为锚记，俗称 id 定位，herf="#id"
<a href="#mubiao">点击此处到目标位置</a>
<br /><br />
<div id="mubiao">这里是目标的位置</div>
```

# nav

标签定义导航链接的部分。

```html
<nav>
  <a href="index.asp">Home</a>
  <a href="html5_meter.asp">Previous</a>
  <a href="html5_noscript.asp">Next</a>
</nav>
```
