# 快捷键

```yaml
win+r cmd: 命令提示符
alt+空格: 窗体程序菜单栏
```

## 运行


```s
calc #计算器
mspaint #画图
notepad #记事本
explorer # 资源管理器
dvdplay # DVD播放器
taskmgr   		#打开任务管理器
gpedit.msc		#打开本地组策略编辑器
services.ms	#打开服务
regedit        #打开注册表
winver			#检查Windows版本
mstsc 			#远程桌面连接
mmc 			#打开控制台
chkdsk.exe     #打开Chkdsk磁盘检查
osk			#打开屏幕键盘
```
