# 1.密码学基础

比特币，加密货币`crypto-currency`，密码学原理：hash与签名

hash函数的性质：

```
1. collision resistance碰撞抵抗，两个不同的数的hash值是不相同的，目前，MD5已经被证实是有可能相同的
2. hiding hash函数是单向的，输入空间足够大，且分布均匀 ：输入+随机数norce生成哈希，使分布均匀
3. puzzle friendly :比特币工作量证明，挖矿很难，验证很容易
```
